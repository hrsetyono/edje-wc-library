Collection of code to help developers customize WooCommerce site.

== Description ==
WooCommerce is a fantastic feature-packed platform, but complicated to use. This plugins brings order to that.

Once Edje Woo is installed and activated in your plugin directory, it gives any WooCommerce theme the ability to take advantage of our features.

Visit our [Documentation](https://github.com/hrsetyono/edje-woo) to get started.

**Note**: This plugin is still in Beta, use at your own risk.
